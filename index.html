Metodologías de Aprendizaje

En este curso utilizaremos metodologías activas para mejorar la retención del conocimiento.

Flipped Classroom (Aula Invertida)

El material se facilita previamente. Los alumnos revisan la teoría en casa y usamos el tiempo de clase para resolver dudas y hacer prácticas.

Puzzle de Aronson (Jigsaw)

Técnica de aprendizaje cooperativo donde cada alumno se vuelve experto en un tema:

  1. Formación de grupos base.
  2. División del material (HTML, CSS, JS, Diseño).
  3. Reunión de expertos.
  4. Regreso al grupo base para enseñar al resto.

HTML5 (HyperText Markup Language)

HTML es el lenguaje de marcas estándar para la creación de páginas web. Define la estructura básica.

Reglas fundamentales:

  • Todo debe estar dentro de etiquetas.
  • Las etiquetas deben anidarse correctamente.
  • Existe una cabecera (<head>) y un cuerpo (<body>).

CSS (Cascading Style Sheets)

CSS se encarga de la decoración y el aspecto visual de la estructura HTML. Permite separar el contenido de la presentación.

Con CSS podemos cambiar:

  • Colores y fondos.
  • Tamaños de letra (Fuentes).
  • Márgenes y alineación (Modelo de caja).

JavaScript

JavaScript es un lenguaje de programación que se ejecuta en el navegador del cliente. Proporciona interactividad a las páginas web.

Funciones comunes:

  • Validar formularios antes de enviarlos.
  • Modificar el CSS en tiempo real (eventos).
  • Ocultar o mostrar elementos dinámicamente.
