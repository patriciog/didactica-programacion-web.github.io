3.2 Operadores y expresiones

La sintaxis incluye una amplia gama de operadores para manipular estos datos:


Aritméticos y Asignación:
Soportan las operaciones matemáticas clásicas (+, -, *, /, módulo % y potencia **), incrementos/decrementos (++, --) y formas abreviadas de asignación como += o *=. El operador + también se utiliza para concatenar (unir) cadenas de texto.


Comparación:
JS ofrece comparaciones débiles (==, !=) y comparaciones estrictas (===, !==). Las estrictas son especialmente importantes porque verifican no solo que el valor sea igual, sino que el tipo de dato también coincida.


Lógicos:
Permiten encadenar condiciones usando && (AND), || (OR) y negar expresiones con ! (NOT).
