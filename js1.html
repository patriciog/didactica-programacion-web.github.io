3.1 Variables, Constantes y tipos de Datos

Las variables actúan como contenedores para los valores. Dado el tipado dinámico de JS, no es necesario declarar el tipo de dato al crearlas. Se declaran con la palabra clave let (ej. let nombre = valor;).

Las constantes se declaran con const y su valor no puede ser alterado posteriormente. Los nombres de estas variables distinguen entre mayúsculas y minúsculas, deben empezar por una letra (evitando guiones bajos al inicio) y no pueden utilizar palabras reservadas del lenguaje como function, var o for.

JS maneja varios tipos de datos primitivos y estructurados. Primitivos: Incluyen números (number) ya sean enteros o decimales, cadenas de texto (string) delimitadas por comillas simples o dobles, y booleanos (boolean) que representan true o false.

Arrays (Matrices): Colecciones de datos ordenados que se escriben entre corchetes [] (ej. let lista = ). Se accede a ellos mediante su posición, empezando por el índice 0.

Objetos: Estructuras que almacenan datos en forma de pares de propiedad: valor y se encierran entre llaves {}. Se accede a sus propiedades utilizando la notación de punto (ej. perro.nombre).
