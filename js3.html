3.3 Estructuras de Control de Flujos

Para dirigir la lógica del programa, JavaScript utiliza estructuras de control de flujo. Las sintaxis es la siguiente:

Condicionales: Bloques if...else para bifurcaciones, declaraciones switch para múltiples casos, y el operador ternario (condición) ? true : false para decisiones en una sola línea.

Bucles (Iteraciones): Estructuras como for (y su variante for...of para recorrer arrays), while y do...while que permiten ejecutar un bloque de instrucciones múltiples veces.

Funciones: Al ser un lenguaje funcional, las funciones son elementos de primer orden. Se declaran con la palabra clave function seguida de su nombre y paréntesis para los parámetros (datos adicionales que necesita para trabajar). Pueden devolver resultados usando la sentencia return. Un rasgo útil de su sintaxis es que permite definir parámetros opcionales asignándoles un valor por defecto en la propia declaración (ej. function saludar(nombre = "usuario")).
