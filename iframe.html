¿Qué es un Iframe?

Un iframe se utiliza para mostrar una página web dentro de otra página web. Es muy útil para incrustar vídeos, mapas o documentos externos sin que el usuario tenga que abandonar tu sitio.

1. Sintaxis y Atributos Básicos

Un iframe se define con la etiqueta <iframe>. Sus atributos más importantes son:

  • src: Especifica la URL (dirección web) de la página que se va a mostrar dentro del marco.
  • width y height: Definen el ancho y el alto del marco (por defecto en píxeles, aunque aceptan porcentajes).
  • style: Por defecto, los iframes tienen un borde. Para quitarlo o cambiarlo, puedes usar la propiedad CSS border (ej. style="border:none;").
<iframe src="pagina_externa.html" width="100%" height="300" style="border:none;"> </iframe>

2. Iframe como destino de un enlace

Una característica muy potente es usar un iframe como "marco de destino" para que, al hacer clic en un enlace de tu página, este se abra dentro del propio iframe y no en una pestaña nueva.

Para lograrlo, el atributo target del enlace debe tener el mismo valor que el atributo name del iframe:

<iframe src="inicio.html" name="marco_ejemplo"></iframe> <p> <a href="http://www.w3schools.com" target="marco_ejemplo">Abrir W3Schools</a> </p>
