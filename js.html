3. Introducción a JavaScript

JavaScript es un lenguaje de programación de secuencias de comandos, que permite crear contenido dinámico, controlar multimedia y manipular la página web sin necesidad de un compilador. Es un lenguaje funcional, de tipado dinámico y orientado a objetos basado en prototipos.

Siguiendo con el simil de la casa, el JavaScript sería la parte de domótica que añadida a una casa, la hace ser una casa inteligente.

El código JavaScript se integra en el documento HTML de forma similar a CSS, ya sea internamente entre etiquetas <script> o externamente haciendo referencia a un archivo (ej. <script src="script.js">). Para documentar y explicar el código, la sintaxis permite usar "//" para comentarios de una sola línea y "/* ... */" para comentarios de bloque.
